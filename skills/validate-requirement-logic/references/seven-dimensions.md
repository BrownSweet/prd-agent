# 七维扫描清单

| 维度 | 最低检查项 |
|---|---|
| 依赖关系 | 前置能力、调用顺序、删除传播、循环依赖 |
| 状态流转 | 初态、所有中间态、终态、触发者、回退 |
| 边界条件 | 空值、最小值、最大值、重复、并发、幂等 |
| 操作影响 | 历史数据、缓存、外部展示、通知、生效时间 |
| 逆向操作 | 取消、删除、恢复、补偿、退款、审计 |
| 交互一致性 | 反馈载体、返回目标、确认规则、禁用态 |
| 异常场景 | 网络错误、超时、无权限、数据失效、依赖故障 |

## 关闭规则

- 阻断问题只能在需求中出现明确解决方案后标记 `resolved`。
- 重要问题可解决或由用户提供理由后标记 `waived`。
- 建议问题允许保持开放，不阻断后续流程。
- 不得因为用户说“下一步”而更改问题状态。
