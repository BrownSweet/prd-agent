# 产品类型适配

| 类型 | 典型特征 | 非功能重点 |
|---|---|---|
| A 管理后台 | 表格、配置、审核、运营、权限 | RBAC、审计、批量操作、响应时间 |
| B C端应用 | 用户旅程、转化、移动端、内容消费 | 首屏、兼容性、弱网、可访问性 |
| C API/服务 | 接口、调用方、SDK、微服务 | SLA、限流、幂等、版本兼容 |
| D 数据产品 | 指标、维度、报表、看板 | 口径、时效、权限、导出 |
| E 中台/平台 | 多租户、能力复用、策略、工作流 | 隔离、扩展性、治理、稳定性 |
| F 硬件/嵌入式 | 设备、固件、传感器、连接 | 功耗、离线、升级、故障恢复 |

复合产品以主类型决定文档重心，并把次类型特有要求合入对应章节。
